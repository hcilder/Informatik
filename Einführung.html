Einführung

Installation

Sie werden auf Ihrem eigenen Gerät mit lokalen Dateien arbeiten. Es ist wichtig, dass Sie einen Ordner auf Ihrem Gerät anlegen, in welchem Sie alle für Ihr Projekt benötigten Dateien speichern!
  • Laden Sie zunächst den VS Code Texteditor herunter und installieren Sie ihn auf Ihrem Gerät.
  • Zudem benötigen Sie den Webbrowser Chrome.

VS Code Plugins

Wenn Sie VS Code heruntergeladen haben, müssen Sie noch einige Plugins installieren. Dazu Öffnen Sie VS Code, drücken CTRL + P und geben den folgenden Code, jeweils Zeile für Zeile, ein: 
ext install ritwickdey.LiveServer
ext install esbenp.prettier-vscode

Arbeitsweise

Auf der Startseite finden Sie Theorie zu HTML und CSS. Ihnen stehen jeweils eine PDF Datei mit Theorie sowie ein Link mit weiteren Informationen zur Verfügung. Innerhalb der vier Doppellektionen sollten Sie folgende Tags in Ihre Webseite einbauen:
  • HTML: Nutzen Sie die Tags <title>, <h1>, <h2>, <p> und <br>.
  • HTML: Verwenden Sie mindestens drei Textformatierungen.
  • HTML: Fügen Sie mindestens ein Bild und einen Link ein.
  • HTML: Erstellen Sie eine Liste mit mindestens zwei Einträgen.
  • CSS: Verwenden Sie den Universal-, den Element- und den Class- Selektor.
  • CSS: Nutzen Sie das Box-Modell.

Projekt erstellen

Laden Sie die Vorlagen für Ihr Projekt herunter und speichern Sie diese auf Ihrem Gerät. Alle weiteren Dateien, welche Sie für Ihre Webseite verwenden, sollten im selben Ordner gespeichert werden!
