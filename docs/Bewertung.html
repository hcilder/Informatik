Notenzusammensetzung

Sie finden hier einen Bewertungsbogen für Ihr Projekt. Damit wird Ihre Webseite benotet.
Im Fokus stehen

  • Ihre selbstständige Arbeit,
  • die Verwendung der geforderten Strukturen
  • sowie die Dokumentation des Arbeitsprozesses.

Das Einhalten der Abgabefrist sowie das Format wird ebenfalls miteinbezogen.
Der Inhalt ihrer Webseite sollte sinnvoll strukturiert sein, wird aber nicht bewertet.

Es findet im Zeitlich vorgegebenen Rahmen eine unangekündigte kurze Multiple-Choice Prüfung statt.
Diese wird auf Wissen über HTML und CSS eingehen.
