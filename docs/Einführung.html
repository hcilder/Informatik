Installation

Sie werden auf Ihrem eigenen Gerät mit lokalen Dateien arbeiten. Es ist wichtig, dass Sie einen Ordner auf Ihrem Gerät anlegen,
in welchem Sie alle für Ihr Projekt benötigten Dateien speichern!

  • Laden Sie zunächst den VS Code Texteditor herunter und installieren Sie ihn auf Ihrem Gerät.
  • Zudem benötigen Sie den Webbrowser Chrome.

Arbeitsweise

Auf der Startseite und auf Teams finden Sie Theorie zu HTML und CSS.
Ihnen stehen eine PDF Datei mit Theorie sowie zwei Links mit weiteren Informationen zur Verfügung.
Innerhalb der vier Doppellektionen sollten Sie folgende Tags in Ihre Webseite einbauen:

  • HTML: Nutzen Sie die Tags <title>, <h1>, <h2>, <p> und <br>.
  • HTML: Verwenden Sie mindestens drei Textformatierungen.
  • HTML: Fügen Sie mindestens ein Bild und einen Link ein.
  • HTML: Erstellen Sie eine Liste mit mindestens zwei Einträgen.
  • CSS: Verwenden Sie den Universal-, den Element- und den Class- Selektor.
  • CSS: Nutzen Sie das Box-Modell.

Nutzen Sie am Ende jeder Doppellektion die Kommentarfunktionen für HTML und CSS,
um den Arbeitsprozess innerhalb Ihrer HTML und CSS Datei zu dokumentieren!

Projekt erstellen

Laden Sie die Vorlagen für Ihr Projekt herunter und speichern Sie diese auf Ihrem Gerät.
Alle weiteren Dateien, welche Sie für Ihre Webseite verwenden, sollten im selben Ordner gespeichert werden!
